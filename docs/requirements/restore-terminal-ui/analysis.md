# 需求分析

## 需求背景

用户需要恢复 Cute 终端的界面功能，包括：
- 输入框
- 侧边栏
- 设置页面
- 其他终端核心界面

**不需要恢复的功能**：
- 云同步功能
- AI 助手功能
- Agent 功能
- SSH/远程终端

## 当前状态

### 已删除的核心文件

原始 Warp 项目有 606 个 terminal 模块文件，当前只剩 6 个空壳文件：

| 文件 | 原始行数 | 当前状态 |
|------|---------|---------|
| `terminal/view.rs` | 27,635 | 已删除 |
| `root_view.rs` | 3,610 | 已删除 |
| `terminal/input.rs` | ~5,000 | 已删除 |
| `terminal/model/` | ~10,000 | 已删除 |

### 保留的基础设施

- `cute_terminal` crate：终端模拟核心（grid、cell、ANSI 解析等）
- `cuteui` crate：UI 框架（elements、rendering、windowing）
- `cuteui_core` crate：UI 核心类型和 traits

## 恢复方案

### 方案一：从 Warp 恢复核心文件（推荐）

**优点**：
- 功能完整，界面成熟
- 已经过大量测试验证
- 用户体验一致

**缺点**：
- 需要处理大量依赖
- 需要移除 AI/云相关代码
- 工作量较大

**恢复范围**：

1. **终端核心模块**（必须恢复）
   - `terminal/model/` - 终端状态模型
   - `terminal/input.rs` - 输入处理
   - `terminal/event.rs` - 事件处理
   - `terminal/grid_renderer.rs` - 网格渲染
   - `terminal/block_list_element.rs` - 块列表元素

2. **终端视图模块**（必须恢复）
   - `terminal/view.rs` - 终端视图（需移除 AI 部分）
   - `terminal/view/ambient_agent/` - 移除或简化

3. **根视图**（必须恢复）
   - `root_view.rs` - 主窗口视图（需移除云/AI 部分）

4. **设置模块**（必须恢复）
   - `settings/` - 设置系统
   - `terminal/general_settings.rs` - 通用设置
   - `terminal/keys_settings.rs` - 快捷键设置

5. **侧边栏**（必须恢复）
   - `pane_group/` - 面板组
   - `workspace/` - 工作区

6. **不需要恢复**
   - `ai/` - AI 功能
   - `auth/` - 认证功能
   - `cloud_object/` - 云对象
   - `drive/` - 云盘
   - `server/` - 服务端

### 方案二：新建简化版终端界面

**优点**：
- 代码简洁，无历史包袱
- 可以按需添加功能
- 工作量可控

**缺点**：
- 功能有限
- 需要从零实现
- 用户体验可能不如 Warp 成熟

**实现范围**：
1. 创建新的 `TerminalView` 使用 `cute_terminal` crate
2. 实现基本的输入框和命令执行
3. 实现简单的设置面板
4. 无侧边栏，单窗口模式

## 恢复策略

**从原始 Warp 代码恢复所有终端界面功能，只移除 AI 和云同步相关代码。**

### 需要恢复的完整功能

1. **终端核心**
   - 输入框和命令执行
   - 终端输出显示
   - 命令历史
   - 自动补全

2. **界面布局**
   - 左侧侧边栏（文件浏览器、工作区）
   - 右侧侧边栏（diff 功能，可收起）
   - 终端头部（当前目录、Git 分支）
   - 终端底部（命令状态、执行时间）
   - Tab 标签页

3. **设置系统**
   - 外观设置（主题、字体、颜色）
   - 快捷键设置
   - Shell 设置
   - 窗口设置

4. **其他功能**
   - 搜索功能
   - 复制粘贴
   - 分屏功能

### 需要移除的功能

- AI 助手相关代码
- 云同步相关代码
- Agent 相关代码
- SSH/远程终端相关代码

## 用户确认的需求

### 界面功能
- **左侧侧边栏**：文件浏览器、工作区切换
- **右侧侧边栏**：类似 IDEA 的 diff 功能，可以收起
- **终端头部**：显示当前目录、Git 分支等信息
- **终端底部**：显示命令状态、执行时间等信息
- **设置页面**：
  - 外观设置（主题、字体、颜色）
  - 快捷键设置
  - Shell 设置（Shell 选择、环境变量）
  - 窗口设置（窗口行为、启动选项）

### 不需要的功能
- 云同步功能
- AI 助手功能
- Agent 功能
- SSH/远程终端
